# 典型应用场景

## 1. 满意度驱动分析

用户对产品的整体满意度（DV）受哪些子维度（IV）驱动？

**典型结构**：
- DV：整体满意度 / NPS / 推荐意愿
- IV：价格、质量、服务、品牌、易用性等子维度评分

**输出解读**：权重最高的是核心驱动因素，即使其简单相关系数不是最高的。

## 2. 体验度量体系

例如抖音图文内容满意度研究：

| 变量 | 含义 |
|------|------|
| total | 整体满意度（DV） |
| sub1 | 推荐符合喜好 |
| sub2 | 内容丰富质量高 |
| sub3 | 操作流畅清晰 |
| sub4 | 评论讨论氛围好 |

**结果示例**（N=865, R²=0.606）：
- sub2 内容丰富质量高：27.6%
- sub1 推荐符合喜好：27.5%
- sub3 操作流畅清晰：24.6%
- sub4 评论讨论氛围好：20.3%

**洞察**：虽然 sub4 与 DV 的简单相关（0.66）不是最高，但 RWA 控制了共线性后揭示了各因子的真实独立贡献。

## 3. 二级指标满意度计算

当需要将三级指标的 RWA 权重汇总到二级指标时：

```
二级指标满意度 = Σ(wt_j × satisfaction_j) / Σ(wt_j)
```

其中 wt_j 是三级指标的标准化相对权重（εj / R²）。
